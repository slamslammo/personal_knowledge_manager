# 深度学习 · 知识梳理

一份系统讲解深度学习原理的知识库:从数学基础,到线性回归、分类、感知机、卷积与循环网络,直至注意力与 Transformer。每个知识点力求给出**直觉、严谨公式与可视化**三者并重。

左侧为知识结构,选择任意分支展开,再选择知识点阅读对应讲解。

- 📚 **参考资料**:精选学习材料(教材、视频、可视化、论文实现)
- 🧮 **数学基础**:深度学习所需的微积分、线性代数、概率核心
- 🤖 **机器学习基础**:线性模型(回归、分类)与"训练"框架
- 🧠 **深度学习**:多层神经网络(MLP、CNN、RNN、注意力 / Transformer)
- 🧪 **实验主线**:从零实现到 mini-GPT 的渐进式实践
