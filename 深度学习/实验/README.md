# 实验

把知识库里的概念做成**可运行、可复现、可改造**的小项目:每个子目录自带数据与代码,能直接在 Jupyter / VS Code 跑起来。定位是"内容轨"之外的**动手轨**——读懂概念后在这里上手验证。

## 约定

- 每个子项目自带:`README.md`(目的 / 运行 / 产物)、代码、所需数据;产物图写入各自的 `figures/`。
- 代码用 `# %%` 分单元,既能 `python xxx.py` 直跑,也能在 Jupyter / VS Code 逐块交互运行。
- 与知识库对应的图,**由本项目代码生成、数据用同一份 csv**,保证文档与项目一致、可复现。

## 子项目

- **[MLP非线性可视化/](MLP非线性可视化/)** —— 配套 [04-深度学习 · MLP 引入非线性的作用](../知识库/04-深度学习/01a-MLP引入非线性的作用.md)。50 点环形数据、numpy 手写 MLP:线性局限 → 手工 r² 降维 → 通用 tanh 升维(真实 (h₁,h₂,h₃) + 切面 GIF)→ 加大宽度逼近圆。**该知识点的全部图都由它生成。**
- **实验主线:从零搭建 mini-GPT(规划中)** —— 线性回归 → softmax → MLP → CNN → 字符级语言模型 → mini-GPT,层层递进。
