# 深度学习

这是 `personal_knowledge_manager` 的第一个知识管理实验板块。目标是用 AI 协作方式系统梳理深度学习:从数学基础、机器学习基础,到 MLP、反向传播、CNN、RNN、注意力与 Transformer,并逐步通向 mini-GPT 实验主线。

## 阅读入口

下载仓库后,直接用浏览器打开:

[知识地图.html](知识地图.html)

该 HTML 使用本目录内的相对路径引用图片和内容,不依赖作者本机绝对路径。

## 目录结构

```text
深度学习/
├── README.md
├── 知识库/          # 内容源头:一个知识点一个 md
├── 实验/            # 可运行、可复现的小实验
├── assets/          # 正文图片与 GIF
├── build.py         # 生成 HTML 阅读视图
└── 知识地图.html    # 浏览器阅读入口
```

## 维护方式

修改 `知识库/` 后,在本目录运行:

```bash
python3 build.py
```

然后重新打开或刷新 `知识地图.html`。

源 Markdown 中的图片和本地文件链接按源文件所在目录写相对路径;生成 HTML 时,`build.py` 会把开头的 `../` 归一化为板块目录下的路径,以兼顾 GitHub 源文件浏览和本地 HTML 阅读。
